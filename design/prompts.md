## Prompty do generowania szkiców i prototypów

Poniżej znajdziesz gotowe prompt-y (PL + EN) do użycia w generatorach obrazów / narzędziach do prototypowania (Midjourney, Stable Diffusion, Figma Plugins itp.) oraz prompt do przeglądu tekstów UI, żeby nie brzmiały jak wygenerowane przez model.

1) Lo‑fi wireframes (PL)
"Stwórz serię trzech lo‑fi wireframe'ów aplikacji mobilnej i webowej o nazwie 'Steady Wheel Hub' — ekrany: 1) ekran logowania z polami email/hasło i przyciskiem demo, 2) dashboard z listą pojazdów, nadchodzącymi serwisami i szybkim CTA 'Add Receipt', 3) formularz dodawania rachunku z polem vendor, amount, vehicle select, date i upload zdjęć. Użyj prostych konturów, brak kolorów, wyraźne priorytety informacji, layout responsywny (mobile + desktop)."

2) Hi‑fi mockups (EN)
"Create hi‑fi screens for a vehicle maintenance web app called 'Steady Wheel Hub'. Provide three polished screens: Login, Dashboard (vehicle cards, upcoming services, recent receipts), and Add Receipt (form with photo previews). Use a clean modern UI, neutral background, blue/accent highlights, clear CTAs, accessible typography and visible focus states. Show desktop and narrow‑viewport (mobile) variants. Export as PNG mockups." 

3) User flow visual (PL)
"Wygeneruj diagram przepływu użytkownika (user flow) dla scenariusza: Użytkownik loguje się → widzi dashboard → przechodzi do pojazdów → dodaje rachunek z załączonym zdjęciem → otrzymuje komunikat sukcesu. Zaznacz kroki wymagające uwierzytelnienia i punkty, gdzie może wystąpić błąd sieciowy."

4) Prompt do rewritingu tekstu UI (usuń 'GPT tone') (PL)
"Przejrzyj poniższe fragmenty tekstu interfejsu użytkownika i przepisz je tak, by brzmiały naturalnie, zwięźle i 'ludzkim' tonem — bez wzmianki o asystencie AI, bez sformułowań typu 'Asystent', 'Jako model językowy', 'ChatGPT' itp. Zachowaj sens i długość komunikatów. Tekst: <wklej treść do sprawdzenia>" 

5) Prompt do wygenerowania obrazków przycisków/ikon (EN)
"Generate SVG icon set (24x24) for vehicle app: car, receipt, wrench, calendar, upload, user. Style: rounded stroke, 2px, monochrome, matching modern dashboard UI." 

Użycie
- Skopiuj odpowiedni prompt, podmień nazwy/kolory jeśli chcesz dopasować do appki. Wyniki graficzne dodaj do `design/` jako PNG/SVG i dołącz do README.
